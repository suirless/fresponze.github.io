Инструкция к запуску тестовых программ.

## Подготовка к запуску

Если вы хотите использовать библиотеку Fresponze, то для начала разработки рекомендуется изучить примеры с [GitHub репозитория](https://github.com/suirless/Fresponze/tree/master/examples).

Чтобы собрать тестовые программы, вам следует ввести эту команду:
```sh
cmake .. -G"Your Generator" -BUILD_EXAMPLES=ON
```

После того, как вы собрали примеры, вам следует указать рабочую директорию - рекомендуется указывать директорию исходников текущего примера (пример: `C:/source/Fresponze/examples/custom_callback`)

## Custom Callback

Первый пример - custom_callback. В нем демонстрируется возможность создания собственного колбека для ручного манипулирования с выходным сигналом (в примере показано, как можно генерировать обычную синусоиду через этот колбек). 

Такое применение может пригодится, когда вы делаете собственную DAW или свой аудиоредактор и позволяет использовать Fresponze как портативную библиотеку для вывода звука. Учитывайте, что в этом случае вы должны самостоятельно реализовать ресемплинг либо использовать внутреннюю имплементацию (см. [IBaseResampler](fresponze_resampler)).
